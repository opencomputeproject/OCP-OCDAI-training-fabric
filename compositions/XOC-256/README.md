# XOC‑256 — Overview

Welcome. XOC‑256 composes two OPG‑128 units under a spine layer to form a larger cluster while keeping familiar OPG patterns. Variants mirror OPG‑128 (rail‑optimized and single‑homed). The compositions in this repository are designed to work with air or liquid cooling at any density.

Operator benefits
- Predictable scaling from OPG‑128: wiring and scheduling behaviors remain familiar.
- Better performance with rail‑local placement: jobs kept within an OPG minimize spine load.
